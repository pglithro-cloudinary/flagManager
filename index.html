Flag Helper v0.9

Instructions

  1. Install the bookmarklet
  2. Navigate to the CLOUD settings page you want to copy from
  3. Click on the Bookmarklet and then "Save Current Flags" to capture the current flag settings
  4. Navigate to a different CLOUD settings page. Click on the bookmarklet and then choose "Apply Flags" to compare/apply
  5. Changes made are shown in three ways
    • Shown in Alert at the time you compare
    • Captured in the Admin Log field towards the bottom of the page
    • Flags which were previously checked and now switched off are shown in red, green for the opposite situation.
  6. CAUTION - You can now optionally save the new settings - be very careful, especially if you encounter warnings. If in doubt review by hand before saving.

Known issues: There is a chance that the number/order of flags might change, this could have very unexpected results. To try and combat this I capture the number of flags and warn if it is different to the current number.
